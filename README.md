# Library Management System

This is a Java-based Library Management System designed to automate and simplify common library operations such as book issuing, returning, and user management. It provides a role-based structure for efficient handling of library activities.

## Features
- Role-based access (Admin, Librarian, Student)  
- Add, update, and manage books  
- Borrow and return functionality  
- Due-date tracking for issued books  
- User dashboard for different roles  
- Database-backed storage  

## Tech Stack
- Java (Swing)  
- MySQL  

## Purpose
This project was developed for academic learning to understand how a complete management system can be built using Java with GUI and database integration. It reflects real-world library workflows in a digital form.

## How to Run
1. Clone the repository  
2. Import the project into your IDE  
3. Configure the MySQL database  
4. Run the main class
